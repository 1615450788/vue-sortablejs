# vue-sortablejs

vue下对sortablejs的包装

## 依赖
- Vue 2.0.0+
- sortablejs 1.0.0+

## 安装方式
```
npm install vuejs-sortablejs
```

## 示例：
```vue
<ul v-sortable="option">
    <li>Foo</li>
    <li>Bar</li>
    <li>Baz</li>
</ul>
```

## 大小
- 1k(不含`sortablejs`)

## 参数

- `{Object} option` 对应`sortablejs`的`option`
